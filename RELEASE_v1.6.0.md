# 🎉 WordEasy v1.6.0 - 词库分类管理系统

## 📢 重大更新

本次更新引入了**完整的词库分类管理系统**，这是 WordEasy 有史以来最大的功能升级！

---

## ✨ 新功能亮点

### 🗂️ 多词库管理
- 创建无限个独立词库
- 每个词库独立配置和管理
- 支持7种语言（英语、日语、中文、韩语、德语、法语、西班牙语）
- 3种词库类型（单词库、课程库、自定义）

### 🎯 智能学习设置
- **每日新学数量**：控制学习强度（1-1000个/天）
- **新学批次数量**：优化单次学习时长（1-100个/批次）
- 灵活的学习节奏，适应不同场景

### 🧠 三种记忆算法
1. **FSRS** (推荐)：最先进的自适应算法
2. **StepMaster**：固定间隔，适合初学者
3. **SM-2**：经典SuperMemo算法

### ⚡ 优先级控制
- 新学优先级：默认顺序 / 随机
- 复习优先级：按遗忘曲线 / 随机
- 重复过滤：过滤 / 允许

### 📊 独立统计
- 每个词库单独统计进度
- 每日学习数据记录
- 学习曲线可视化（即将推出）

## 🚀 快速开始

### 1. 数据库迁移

**重要**：升级前请先迁移数据库

```bash
cd backend
python migrations/add_deck_system.py
```

迁移会自动：
- ✅ 创建新的数据库表
- ✅ 创建「默认词库」
- ✅ 迁移所有旧数据
- ✅ 备份原始数据

### 2. 创建第一个词库

```
1. 进入「词库管理」页面
2. 点击「➕ 创建新词库」
3. 填写词库信息：
   - 名称：考研核心词汇
   - 每日新学：50个
   - 批次大小：20个
   - 语言：英语
   - 算法：FSRS
4. 保存并开始添加单词
```

### 3. 导入单词

支持三种方式：
- 📤 文件导入（TXT/CSV/JSON）
- ✏️ 快速添加（单个输入）
- 📋 批量添加（粘贴文本）

## 💡 最佳实践

### 学习量设置建议

| 可用时间 | 每日新学 | 批次大小 |
|---------|---------|---------|
| 10分钟 | 20-30个 | 10-15个 |
| 30分钟 | 50-80个 | 20-30个 |
| 1小时 | 100-150个 | 30-50个 |
| 2小时+ | 200-300个 | 50-100个 |

### 算法选择建议

- 🔰 **新手**：StepMaster（简单直观）
- 🎓 **进阶**：FSRS（最优效果）
- 📚 **传统**：SM-2（稳定可靠）

### 词库分类建议

1. **按难度**：初级、中级、高级
2. **按考试**：CET-4、CET-6、考研、托福
3. **按主题**：生活、工作、学术、旅游
4. **按课程**：教材第1-12章
