# 🎬 词库导入功能演示

本文档通过实例演示 WordEasy v1.5.0 的三种词库导入方式。

---

## 📝 演示 1：快速添加单词

### 场景
你在阅读英文文章时遇到生词 "serendipity"

### 操作步骤
1. 进入「词库管理」页面
2. 在「✏️ 快速添加单词」模块：
   - 英文单词：`serendipity`
   - 中文释义：`意外发现；机缘巧合`
3. 点击「➕ 添加单词」或按回车

### 结果
```
✅ 成功添加单词：serendipity
```

### 适用场景
- ✅ 临时记录生词
- ✅ 快速添加少量单词（1-5个）
- ✅ 课堂笔记记录

## 📋 演示 2：批量添加

### 场景
你整理了一周的学习笔记，有10个生词需要添加

### 准备数据
```
abandon 放弃；抛弃
ability 能力；才能
absolute 绝对的；完全的
abstract 抽象的；摘要
academic 学术的；学院的
accept 接受；承认
access 通道；访问
accident 事故；意外
accompany 陪伴；伴随
accomplish 完成；实现
```

### 操作步骤
1. 复制上面的内容
2. 在「📋 批量添加」文本框中粘贴
3. 点击「📝 批量添加」

### 结果
```
✅ 成功添加 10 个单词
```

### 支持的格式
```
# 格式1：空格分隔（推荐）
word1 释义1
word2 释义2

# 格式2：Tab分隔
word1	释义1
word2	释义2

# 格式3：竖线分隔
word1|释义1
word2|释义2
```

## 🔄 演示 4：在线格式转换

### 场景
你有一个 CSV 格式的词库，需要转换后导入

### 原始数据（CSV格式）
```csv
hello,你好
world,世界
computer,计算机
program,程序
database,数据库
```

### 操作步骤

#### 步骤 1：显示转换器
1. 在「📤 导入词库文件」模块找到「💡 格式不匹配？」
2. 点击「显示转换器」

#### 步骤 2：配置转换
1. 输入格式：选择「CSV (逗号分隔)」或「自动检测」
2. 粘贴上面的 CSV 内容到文本框

#### 步骤 3：执行转换
1. 点击「🔄 转换格式」
2. 查看转换结果：
```
hello|你好
world|世界
computer|计算机
program|程序
database|数据库
```

#### 步骤 4：使用结果
选择以下操作之一：
- 「📋 复制结果」：复制到剪贴板，保存为TXT文件
- 「⬆️ 直接上传」：立即导入到词库

### 支持的输入格式

#### JSON 数组格式
```json
[
  {"word": "hello", "meaning": "你好"},
  {"word": "world", "meaning": "世界"}
]
```

#### JSON 对象格式
```json
{
  "hello": "你好",
  "world": "世界"
}
```

#### CSV 格式
```csv
hello,你好
world,世界
```

#### TXT 格式（空格分隔）
```
hello 你好
world 世界
```

## 🎯 完整工作流演示

### 场景：准备考研词汇学习

#### 第1步：下载词库
从 GitHub 下载 KyleBing 的考研词汇：
```bash
git clone https://github.com/KyleBing/english-vocabulary.git
```

#### 第2步：使用命令行工具转换（可选）
```bash
cd wordeasy/tools
python convert_vocab.py ../../english-vocabulary/5\ 考研-乱序.txt kaoyan.txt
```

输出：
```
============================================================
WordEasy 词库转换工具
============================================================
输入文件: 5 考研-乱序.txt
输出文件: kaoyan.txt
检测格式: TXT
============================================================

✓ 转换完成！
  成功转换: 5500 个单词
  输出文件: kaoyan.txt
============================================================
```

#### 第3步：导入到系统

**方式A：直接上传原文件**
1. 在网页中进入「词库管理」
2. 上传 `5 考研-乱序.txt`（系统自动识别格式）

**方式B：使用在线转换器**
1. 打开原文件，复制内容
2. 使用网页的「在线格式转换器」
3. 点击「⬆️ 直接上传」

**方式C：上传转换后的文件**
1. 上传 `kaoyan.txt`

#### 第4步：验证导入
查看「📊 词库统计」：
```
初级单词: 1500
中级单词: 2800
高级单词: 1200
总单词数: 5500
```

#### 第5步：开始学习
1. 返回首页
2. 选择「初级」难度开始学习

4. 点击「📝 批量添加」

## 🐛 常见问题处理演示

### 问题 1：中文乱码

#### 现象
导入后显示：`鎴戜滑`

#### 原因
文件编码不是 UTF-8

#### 解决步骤
1. 用记事本打开文件
2. 另存为 → 编码：UTF-8
3. 重新上传

### 问题 2：格式识别错误

#### 现象
```
✗ 错误：未找到有效的单词
```

#### 解决步骤
1. 点击「显示转换器」
2. 手动选择正确格式
3. 转换后直接上传

### 问题 3：批量添加失败

#### 现象
```
❌ 未找到有效的单词，请检查格式
```

#### 检查清单
- ✅ 每行只有一个单词
- ✅ 单词和释义有明确分隔
- ✅ 没有空白行
- ✅ 格式一致

#### 修正示例

**错误**：
```
hello你好
world世界
```

**正确**：
```
hello 你好
world 世界
```
